Nombre Proyecto

Entorno de Investigación Aplicada Dinamica Digital

Objetivos
  • La creación de un complemento prototipo para ELGG que permita la confección de planillas basadas en arquetipos.
  • La implementación de una interfaz que permita el arrastre de arquetipos al documento principal para crear una plantilla base de investigación aplicada basadas en las estructuras del proceso científico .
  • La implementación de una interfaz de edición de texto (que tiene como base la plantilla creada anteriormente) basada en bloques de edición (similar a Gutenberg en Wordpress).
Descripción Resumida

Es un entorno digital estructurado en base al proceso científico para crear, escribir, editar, guardar y exportar artículos científicos. Este entorno digital integra en tiempo real los componentes esenciales para un proceso científico cuantitativo, cualitativo o ambos juntos conocido como metodología mixta (números y textos).

Es una herramienta para la creación, edición y exportación de documentos que integra plantillas dinámicas para la elaboración de artículos científicos. El investigador tiene la posibilidad de interactuar con los diversos elementos constitutivos de un proceso de investigación, los cuales se integran para dar el formato y las orientaciones básicas del diseño metodológico seleccionado. De esta forma el proceso de elaboración del documento científico es más amigable y accesible para personas menos experimentadas y optimiza el proceso de aquellos usuarios expertos.

Resultado Esperado

El complemento permite la creación de plantillas base de investigación, las cuales pueden ser editadas mediante el uso de bloques de edición. El documento final creado a partir de la combinación de bloques puede ser exportado como .ODT, .PDF y .JSON para ser utilizado en otros editores y/o plataformas.

Tecnologías involucradas
  • PHP7+
  • HTML5 - CSS3
  • Javascript
  • MongoDB o alternativa para BD’s NoSQL.
Criterios de aceptación
  • El documento creado se puede descargar como formato abierto ODT.
  • El documento creado se puede exportar como PDF.
  • El documento creado (sin estilos, sólo contenido) se puede descargar como JSON.
  • (Opcional) Los documentos se pueden editar en conjunto (edición compartida similar a Google Drive)
  • (Opcional) Integración con Zotero o alternativa para la inclusión de citas.
